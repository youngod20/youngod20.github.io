* pwd: print work directory 현재 작업중인 디렉토리 경로
* ls: lsit segments 파일과 디렉터리 정보제공 (ls -al 세부정보 표시)
* cd: change directory 디렉토리 이동 (cd .. 이전으로 이동)
* mkdir: make directories 디렉토리 생성
* rmdir: remove directories 디렉토리 삭제
* touch: 빈 파일 생성
* cat: 파일 출력
* mv: move 파일 이동
* echo: 큰 텍스트 출력
* sudo: root 명령
* tar: 파일 압축
